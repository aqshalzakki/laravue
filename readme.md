# Vue 3 powered with Laravel token based authentication, Sanctum

## Frontend
1. `cd ./frontend`
2. `npm install`
3. `npm run dev`
4. Open at `http://localhost:5173/`

## Backend
1. `cd ./backend`
2. `composer install`
3. `php artisan key:generate`
4. `php artisan serve`
5. Server at `http://localhost:8000/`
